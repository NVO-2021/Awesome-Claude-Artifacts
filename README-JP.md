## Claudeアーティファクトとは？

Claudeアーティファクトは、ClaudeAIアシスタントによって生成された独立したコンテンツの断片です。これらはコードスニペット、ドキュメント、SVG画像、図表など多岐にわたります。アーティファクトはClaudeの創造性と問題解決能力を示しています。

## このプロジェクトの目的は？

このプロジェクトは以下を目的としています：

1. Claudeの能力と多様性を紹介する
2. 開発者とAI愛好家にインスピレーションを提供する
3. AI生成コンテンツの共有と探索のためのコミュニティを作る

## 貢献方法

私たちは貢献を歓迎し、奨励しています！興味深いClaude生成アーティファクトをお持ちの場合は、以下の手順に従ってイシューとして提出してください：

1. このプロジェクトの[イシューページ](https://github.com/NVO-2021/awesome-claude-artifacts/issues)に行く
2. "New Issue"ボタンをクリックする
3. "Artifact Submission"テンプレート（利用可能な場合）を選択する
4. 必要な情報を記入する：
    - アーティファクトのタイトル
    - 簡単な説明
    - アーティファクトを生成するのに使用したプロンプト（可能な場合）
    - アーティファクトの目的または潜在的な応用
    - アーティファクトの内容（コード、テキスト、画像リンクなど）
5. イシューを提出する

メンテナーチームがあなたの提出を確認し、プロジェクトの基準を満たしている場合は、適切なカテゴリーに追加します。

注意：アーティファクトを提出することで、あなたはそれをこのプロジェクトのオープンソースライセンスの下に置くことに同意したことになります。

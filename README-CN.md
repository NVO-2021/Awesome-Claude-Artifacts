## 什么是Claude人工制品？

Claude人工制品是由Claude AI助手生成的独立内容片段。这些可以是代码片段、文档、SVG图像、图表等。人工制品展示了Claude的创造力和解决问题的能力。

## 这个项目的目的是什么？

这个项目旨在：

1. 展示Claude的能力和多样性
2. 为开发者和AI爱好者提供灵感
3. 创建一个分享和探索AI生成内容的社区

## 如何贡献

我们欢迎并鼓励贡献！如果你有一个有趣的Claude生成的人工制品，请按照以下步骤通过issue提交：

1. 前往本项目的[Issues页面](https://github.com/NVO-2021/awesome-claude-artifacts/issues)
2. 点击"New Issue"按钮
3. 选择"Artifact Submission"模板（如果有的话）
4. 填写必要的信息，包括：
   - 人工制品的标题
   - 简短描述
   - 用于生成人工制品的提示（如果可能的话）
   - 人工制品的用途或潜在应用
   - 人工制品的内容（代码、文本、图像链接等）
5. 提交issue

我们的维护团队将审核你的提交，如果符合项目标准，我们会将其添加到适当的类别中。

注意：通过提交人工制品，你同意将其置于本项目的开源许可下。

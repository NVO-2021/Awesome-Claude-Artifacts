## 什么是Claude人工制品？

Claude人工制品是由Claude人工智能助手生成的独立内容片段。这些可以是代码片段、文档、SVG图像、图表等。人工制品展示了Claude的创造力和解决问题的能力。

## 为什么要做这个项目？

本项目旨在：

1. 展示Claude的能力和多功能性
2. 为开发者和人工智能爱好者提供灵感
3. 创建一个分享和探索人工智能生成内容的社区

## 如何贡献

我们欢迎并鼓励贡献！如果您有一个有趣的Claude生成的人工制品,请按照以下步骤通过问题提交：

1. 前往本项目的[问题页面](https://github.com/NVO-2021/awesome-claude-artifacts/issues)
2. 点击"新建问题"按钮
3. 选择"人工制品提交"模板（如果有的话）
4. 填写必要的信息,包括：
    - 人工制品的标题
    - 简短描述
    - 用于生成人工制品的提示（如果可能的话）
    - 人工制品的用途或潜在应用
    - 人工制品的内容（代码、文本、图片链接等）
5. 提交问题

我们的维护团队将审核您的提交,如果它符合项目的标准,我们将把它添加到适当的类别中。

注意：通过提交人工制品,您同意将其置于本项目的开源许可下。
